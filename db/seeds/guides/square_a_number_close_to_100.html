To square a number in the range 90 to 99:

  • Find the difference of the number from 100
    • Difference is 100 - number
  • Get the last 2 digits by squaring the difference
  • Get the first 2 digits by subtracting the difference from the original number

To square a number in the range 101 to 110:

  • Find the difference of the number from 100
    • For example, use 3 for 103, use 9 for 109
  • Get the last 2 digits by squaring the difference
  • Get the left digits by adding the difference to the original number
