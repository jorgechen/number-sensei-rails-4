Method

This trick works for repeating decimals like where each decimal has two or more repeating digit at the end:

  1. For the numerator, subtract the non-repeating digits from the combined non-repeating and repeating digits
  2. For the denominator:
    • write the same number of 9s as repeating digits
    • then write the same number of 0s as non-repeating digits
