A number is divisible by another number if the remainder is 0. There are quick ways to check if a number is divisible:

A number is divisible by: Example:
2 if the number is even
3 if the sum of the digits is divisible by 3
4 if the last 2 digits is divisible by 4
5 if the last digit is 0 or 5
6 if number is divisible by 2 and 3
8 if the last 3 digits is divisible by 8
9 if the sum of the digits is divisible by 9
10 if last digit is a 0
11 if the difference between step 1 and 2 below is 0:
  1. Starting from the ones digit, add every other digit.
  2. Add the remaining digits.
12 if the number is divisible by 3 and 4
